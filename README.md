# pyscheduler
파이썬기반으로 제작한 스케줄러 프로그램입니다.
동작버전: 3.10.19
exe파일로 사용법 사용하는 라이브러리 다운 및 py버전 설정
uv pip install pyinstaller 로 exe 파일 만들기 라이브러리 다운
assets파일 안 py.ico 밖으로 빼기
flet pack Todo_page.py --icon py.ico 터미널에 입력
이후 exe생성된 곳에 assets 이동
exe생성 이후 바로가기 생성하여 원하는 위치로 이동하여 자유롭게 사용 가능
json파일 저장 위치 C:\Users\<YourName>\Documents\PyScheduler
